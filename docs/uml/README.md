# UML diagrams

This directory stores PlantUML sources and rendered diagrams for Puer Pub.

## Use case diagram

The first diagram describes high-level functional requirements for the tea shop:

- public catalog, cart, checkout, payment, and receipts;
- customer profile, saved addresses, loyalty, and Telegram linking;
- Telegram bot customer flows;
- admin product, order, user, banner, settings, analytics, and Telegram broadcast flows;
- external integrations with Tinkoff, SMS.ru, and Telegram API.

Files:

- [puer-pub-use-case.puml](./puer-pub-use-case.puml) - PlantUML source.
- [puer-pub-use-case.svg](./puer-pub-use-case.svg) - rendered diagram.
